# Миграция с JPremium (CLEARED-1.23.0 и новее)

### ✅ Для каждого игрока переносится:
- Имя в оригинальном регистре
- UUID (uniqueId, не premiumId)
- Пароль (SHA256)
- Статус премиум
- Дата и адрес регистрации
- Дата и адрес последнего входа

:::warning

JPremium странно работает с UUID. Мы переносим только столбец `uniqueId`.

Возможно, вам стоит после переноса включить `force-set-uuid` в `advanced.yml`,
чтобы у игроков всегда принудительно были именно эти UUID, а не стандартные пиратские.
:::

### Как перенестись?

1. Установите последнюю версию BaronessAuth.
2. Расположите оригинальную папку JPremium в plugins/. Jar плагина можно удалить, нужна только папка.
3. Убедитесь, что настройки JPremium корректны. Настройки базы данных должны соответствовать тем, откуда вы хотите перенестись.
4. Введите `/auth migrate JPremium`.
5. Следите за процессом - обращайте внимание на возможные предупреждения.
6. По окончанию процесса будет создан файл `.backup`, содержащий все импортированные данные. Воспользуйтесь `/auth backup load [файл] (по желанию --force)`, чтобы загрузить его.
7. Готово - все игроки перенесены!
